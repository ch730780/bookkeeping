# bookkeeping

מערכת הנהלת חשבונות לעסקים קטנים/פטורים/מורשים.

## טכנולוגיות
- Node.js + TypeScript
- MongoDB
- Express
- Jest (בדיקות)

## פיצ'רים עיקריים
- ניהול הכנסות (קבלות)
- ניהול הוצאות
- ניהול לקוחות, ספקים וקטגוריות
- דוחות הכנסות/הוצאות
- העלאת קבצים (PDF)
- הפקת קבלות PDF

## התקנה והרצה

```bash
git clone <repository-url>
cd bookkeeping
npm install
npm run dev
```

## מבנה תיקיות

- src/models - מודלים של מסד הנתונים
- src/services - לוגיקת שירותים
- src/routes - ראוטים ל-API
- src/controllers - קונטרולרים
- tests - בדיקות
